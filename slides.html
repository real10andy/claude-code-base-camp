PART 1 ・ DIFFERENCE

チャット版AIと Claude Code の違い

07
チャット版AI（ChatGPT 等）
  • 文章を返すだけ。その先は自分でやる
  • 「例を教えて」→ 例文どまり
  • コピペ・保存・送信は手作業
  • 立ち位置：相談相手
できることが段違い
Claude Code
  • PC・Gmail・カレンダーを実際に操作
  • 「返信漏れ探して」→ 自分で見て一覧化
  • 作成・保存・送信まで一気通貫
  • 立ち位置：実働アシスタント
